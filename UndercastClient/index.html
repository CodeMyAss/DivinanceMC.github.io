    • Undercast Client

    Project maintained by UndercastTeam Theme by mattgraham. Edited by goldbattle

    This is a client side mod that is made to enhance your experience on the Overcast Network servers. We have already included great features and look forward to adding many more. Above all we try our best to take what the community wants and make it happen. This is a community made mod and will continue to be so. Feel free to express any and all opinions on it and/or ideas on how to make it better.

    Latest Version: 1.7.9

    Feature List

    • Live ingame stats (use the F6-key for toggleing the ingame display)
    • Custom server menu
    • Custom ingame menu (use the L-key)
    • Ingame configurations (use the P-key)
    • Better observing using full brightness (use G-key)
    • FPS ingame display
    • Current and next map display
    • Team and Server detection
    • Option to filter the [Tip] messages
    • See your total stats
    • get achievements for kills/deaths
    • tracks special kills (e.g. 15000th, 7500th, 3333th)
    • and much more!
