    Installing

    Video Tutorial

    • Download the Undercast Client.

    • Download Minecraft Forge.


    Text Installation Guide
    • Download the latest version of the Undercast Client
    • Download the recommended Minecraft Forge installer. For compatibility check the release notes of the Undercast Client.
    • Run the downloaded installer
    • Select "Install Client" and press OK.
    • Start your Minecraft Launcher. Select the "Forge" profile.
    • Double check the used version using "Edit Profile".
    • Start Minecraft.
    • Click "Edit Profile" in the Minecraft Launcher.
    • Press "Open Game Dir".
    • Put the UndercastClient.jar file into the "mods" folder in the folder which opened.
    • Restart Minecraft.
    • Done!
