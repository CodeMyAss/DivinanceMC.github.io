    About Us

    We are a team of developers who like Minecraft PvP. We especially enjoy the Overcast Network. But there are always some annoying parts (for example keeping track of your stats or checking out what maps are playing on which server while ingame). And as we are developers, we started to code a modification which improves most of these parts. The mod already has a lot of features and it's still growing. If you want, you can checkout and download the mod.
