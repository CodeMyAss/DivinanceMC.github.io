    BorderClans is a Minecraft Bukkit plugin that runs the BorderClans server.

    Latest Version: 1.7.9

    Features

    • Automatic team joining
    • Colored tab-list
    • Colored username/chat messages
    • Developer/Staff/Donor flairs
    • Admin chat (/a)
    • Global chat (/g)
    • Team chat (/t)
    • A currency for achieving certain tasks. (VenomRewards points)
    • A shop to spend your VenomRewards points (/shop)
    • Allows polls to be created by staff members
    • Allows players to vote for polls (/vote
    • Alert staff by reporting players (/report )
    • Check which staff members are online (/staff)
    • Spectator mode
    • And much more...
