    What is Undercast Client v2?

    It is the future of the mod. It's a complete rewrite of the original mod. It allows us to use better coding style and we can give it a better structure. If you are just looking for the released version of the Undercast Mod, you're wrong. But for developers it might be interesting to have a look at it.

    Goals of the rewrite

    • Module system
    • Strict use of Java Coding Conventions
    • Code more understandable
